# blocky-counter
This tool helps calculate how many blocks are necessary to create a cubic structure in a blocky game like Minecraft or Vintage Story.

## Features
- **Accurately calculates blocks needed for a cube home with the option of door size.**
- Automatically formats the amount of blocks to the amount of stacks needed + remainder.
- Input Validation: 
  - Rejects non-integer inputs (floats, text).
  - Strictly rejects negative numbers and zero to ensure valid physical dimensions.
- **Allows users to perform multiple calculations in one session without restarting the program**

## Known Issues
- Currently requires manual input; no GUI interface yet.
